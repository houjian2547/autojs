# autojs
概述：不需要Root权限的JavaScript自动化软件，仅适用于安卓手机。

官网地址：https://hyb1996.github.io/AutoJs-Docs/#/

==================================================================

1.基本信息：

    适配手机品牌：一加5T

    安卓版本：9.0

2.各文件介绍：

    1.main.js为主启动文件。

    2.project.json为打包apk时使用，能指定app名称，版本号等。因为此项目有交互图形，autojs不支持，暂时不能打包使用。
